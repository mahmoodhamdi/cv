# Mahmoud Hamdy Fathy — Portfolio CV

A modern, bilingual (English/Arabic) portfolio CV built with pure HTML, CSS & JavaScript.

## Features

- Dark theme with modern glassmorphism design
- Bilingual support (English ↔ Arabic) with RTL
- Print-ready layout
- Fully responsive
- Smooth animations
- No dependencies — pure HTML/CSS/JS

## Live Preview

🔗 [https://mahmoodhamdi.github.io/cv](https://mahmoodhamdi.github.io/cv)

## Tech Stack

- HTML5, CSS3, JavaScript
- Google Fonts (DM Serif Display, Outfit, JetBrains Mono, Amiri, Noto Kufi Arabic)
